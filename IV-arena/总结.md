## 一、按主题分类

### 1) Go 并发与运行时（最高频）
- GMP：调度机制、阻塞与唤醒、线程协程关系
- Channel：有缓冲/无缓冲、发送接收过程、与 mutex 对比、限流场景
- select/mutex/context：select 随机性、防饿死、mutex 正常/饥饿模式、自旋、context 传值与取消
- 逃逸与 map、切片：闭包逃逸、并发安全

### 2) 数据库与缓存（高频）
- MySQL：B+ 树、索引原理、事务与幻读、唯一索引 vs 普通索引
- Redis：穿透/击穿/雪崩、布隆过滤器、持久化、过期/淘汰策略

### 3) 项目深挖（高频且决定上限）
- 文档中心：选型（ES/Meili/Bleve/MySQL）、倒排、分块与内存优化、AI 搜索兜底
- 云端穿透：NAT/STUN/TURN/ICE、P2P 失败兜底、公网链路与加密
- 相册：模型、DB 字段、人脸与标签流程、模糊/重复检测
- 文件索引同步器：不用 fsnotify 的原因、BFS/DFS、状态机与跨项目复用

### 4) 系统与网络基础（中频）
- Linux 常用：端口排查、kill vs kill -9、shell 基础
- 网络：TIME_WAIT、进程与进程通信、工作目录等

### 5) AI 工程化能力（中频）
- AI 工作流：需求拆解、PRD/方案、代码生成、测试、AI Code Review
- 工具对比：Cursor / Claude Code / Copilot
- 如何用 AI 落地业务：提效比例、质量保障、边界与兜底

### 6) 行为面与协作面（中频）
- 为什么跳槽、看重什么、对公司了解
- 需求协作流程、接口契约、评审与推进能力
