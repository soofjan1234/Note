# Loom Memory
一个背诵系统，类似Anki，但加入AI辅助

## 功能规划

1. 曲线复习
按照动态间隔（类似SM-2算法）+ AI辅助判断难度，每次复习后，根据个性化遗忘曲线，自动调整复习时间

2. 加入
支持设置文件夹 文件
用户可筛选基路径，同时基路径下可以有类似.gitignore的文件，让ai和程序忽略某些文件 
    - 后续功能
    增量索引（避免每次全量扫描）
    文件变更触发更新（类似watch机制）

3. 智能剔除
用户选择丢弃还是留下回答，丢弃也要有个文件记录，后续不会重复加入
遇到某个问题，用户觉得已掌握觉得烦，可以+15天；觉得无用可删除

4. 评分
用户自评：A记住 B模糊 C忘了

AI评分A (全对) + 用户自评 A = 7天 
AI评分A (全对) + 用户自评 B = 4天 
AI评分B (核心正确但细节缺失) + 用户自评 A = 4天 
AI评分B (核心正确但细节缺失) + 用户自评 B = 3天 
AI评分C (完全错误) + 用户自评 B = 2天 
AI评分C (完全错误) + 用户自评 C = 1天 
其它情况，AI自行得出天数

5. 简历、面试
可上传简历以及历次面试txt记录，ai可分析，并更有针对性的出题
